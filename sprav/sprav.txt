- футер
    - набор ссылок
    - переключение языка
    - копирайт

----------------------------------------------------------------

Мои организации

- карточка организации
- интерфейс вкладок
    - вкладка "В справочнике"
        - список организаций
        - диалог удаления организации из списка организаций пользователя
        - диалог удаления организации из базы Справочника
    - вкладка "Заявки на изменение"
        - список заявок
        - диалог удаления заявки
    - вкладка "На подтверждении"
        - список заявок
        - диалог отказа от подтверждения прав и удаления заявки

80 - 100 часов

----------------------------------------------------------------

- список заказов
    - сортировка списка по разным параметрам
    - карточка заказа
    - предпросмотр рекламного объявления
    - диалог доплаты за приоритет
- вкладки
    - вкладка "Активные"
    - вкладка "На проверке"
    - вкладка "Завершенные"
    - вкладка "Архив"
    - вкладка "Все"

60 — 80 часов

Рубрикатор

- интерфейс вкладок
- список рубрик с комментариями
- тултип со списком дополнительных параметров

20 - 40 часов

----------------------------------------------------------------

Главная страница

- диалог с видеоинструкцией
- форма поиска
- презентация приоритетного размещения

40 - 50 часов

----------------------------------------------------------------

Поиск организаций

- форма поиска
    - связка саджестов страны и города
    - кнопка добавления новой организации с подстановкой данных
- список результатов поиска
    - дозагрузка результатов поиска
- диалог подтверждения прав на организацию
- диалог уточнения запроса

80 - 100 часов

----------------------------------------------------------------

Выставление счета

- вкладки выбора счетов по валюте
- список счетов на оплату
- диалог добавления заказов в список на оплату
    - группа чекбоксов с опцией "выбрать все"
- расчет суммы общего счета

20 - 40 часов

----------------------------------------------------------------

Настройки

- форма настроек
    - возможность указать несколько адресов электронной почты
    - выбор типов уведомлений

15 - 20 часов

----------------------------------------------------------------

Добавление организации

- шаг 1 (реквизиты)
    - связка геосаджестов с подстановкой области и района
    - название организации с возможностью указания юр. и англ. названий
    - саджест адресов
    - геокодер
    - ручная установка геометки на карте (API Я.Карт)
    - группа полей для ввода нескольких телефонных номеров
    - поле для указания нескольких сайтов с проверкой доступности урла
    - поле для указания нескольких соцсетей с проверкой доступности урла
    - поле для указания электропочты организации
    - интерфейс для указания времени работы организации
- шаг 2 (виды деятельности)
    - саджест категорий
    - диалог выбора категории из рубрикатора
    - динамически генерируемый набор полей для дополнительных параметров
- шаг 3 (способ размещения)
    - выбор способа размещения: стандарт/приоритет
    - настройка уведомлений
        - диалог изменения адреса электропочты для уведомлений
    - форма создания рекламного объявления
        - счетчики введенных символов
        - динамически генерируемый предпросмотр объявления
        - проверка введенного урла
    - возможность указать дополнительное объявление на другом языке
    - настройки рекламного объявления
        - контрол для выбора даты начала показов
        - расчет стоимости в зависимости от длительности показов
    - форма указания контактной информации рекламодателя
        - поля для ввода контактного лица и телефона
        - группа полей для указания электропочты или факса
    - капча
- валидация

400 - 500 часов
